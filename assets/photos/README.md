## Notes

The website already works without the images. Missing photos show elegant placeholders. Once the photos are uploaded with these exact filenames, they will appear automatically.

For best quality, upload portrait images around 1600px wide or larger. Use JPG for photos and keep each file below 1 MB for fast loading.
