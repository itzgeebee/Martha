# Martha Ifeoma Okadigbo — Birthday Magazine Website

A romantic digital birthday magazine for Martha Ifeoma Okadigbo, built as a static website with a printable A4 magazine version.

## What is included

- Responsive digital magazine homepage
- Premium editorial styling with cream, blush, wine, gold and dark sections
- Full birthday messages from Aaron, Chidera, Mum, Olisa, Chidubem, Kosi, Allison and Ifechukwu
- Gallery sections and photo slots
- Printable layout using CSS `@media print`
- GitHub Pages workflow

## How to add the pictures

Upload the photos into `assets/photos/` using the exact filenames listed in `assets/photos/README.md`.

The site currently shows placeholders where photos are missing. Once the photos are uploaded, they will appear automatically.

## How to print

Open the website in a browser, click **Print Magazine**, then choose:

- Destination: Save as PDF or your printer
- Paper: A4
- Layout: Portrait
- Background graphics: On
- Margins: Default or Minimum

## GitHub Pages

A workflow is included at `.github/workflows/pages.yml`. After GitHub Pages is enabled for the repository, the site should publish automatically from `main`.
